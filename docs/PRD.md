# Product Requirements Document (PRD)

## Problem
Users search for tokens but don't trade, resulting in a low Search-to-Trade conversion rate.

## Goal
Increase Search → Trade conversion.

## Success Metrics
* Conversion Rate
* Trade Volume
* Retention

## Risks
* Fake search spikes
* Bot traffic

## Experiment Design
**Objective:** Evaluate if adding trading signals directly on the Token Page increases the Search-to-Trade Conversion Rate.

* **Variant A (Control):** Current Token Page (Price chart + basic token info).
* **Variant B (Treatment):** Token Page + Trading Signals (e.g., "75% of users bought in last 24h", Social Sentiment Score).

**Measurement:**
* Compare Conversion Lift, Retention Lift, and Trade Volume Lift between the two variants.
